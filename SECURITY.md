# Security Policy

## Reporting a Vulnerability

**Do not open a public issue for security vulnerabilities.**

Please report security vulnerabilities via [GitHub Security Advisories](https://github.com/f0rw4rd/pyprofinet/security/advisories/new).

You will receive a response within 72 hours. If the vulnerability is confirmed, a fix will be released as soon as possible.

## Scope

This policy covers:
- The `profinet-py` Python package published on PyPI
- The source code in this repository
- CI/CD pipeline and build infrastructure

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest  | yes       |
| < latest | no       |
